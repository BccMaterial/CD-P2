**Descrição completa do dataset**  

Origem e credibilidade da base de dados
>O Departamento de Transportes de Minnesota (MnDOT) coleta continuamente esses dados e pesquisadores organizaram a base, que foi então disponibilizada no repositório da UCI.

Como o dataset foi coletado (sensor, organização, técnica etc.)
>Este dataset foi montado a partir de sensores de tráfego instalados na rodovia interestadual I-94 próxima a Minneapolis. 

Tipo de dado coletado (ex.: energia, clima, tráfego, financeiro)
>Volume de tráfego (contagem de veículos por hora), temperatura, condições climáticas, presença de feriados e dados de data/hora.
